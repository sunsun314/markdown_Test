<h1>server.xml 配置</h1>

<h2>整体XML结构</h2>

+ system (目前reload不会做变更)
  - properties 
+ user (可多值,reload可变更)
+ firewall (reload可变更)
  - whitehost
  - blacklist
  - host
+ alarm (reload可变更)

<h2>配置选项</h2>
<h5>注意:"实例/全局属性" 指的是如果集群配置多个中间件,是否要求配置在全局级别保持一致，或者实例间可以不同 </h5>
